The first major hurdle you encounter when learning Linux is actually getting Linux installed on your computer. Installing a completely foreign operating system is daunting, and you are not willing to risk breaking your current Windows installation. Fortunately we can use a virtual machine to learn Linux without touching our current Windows installation.

In this article we'll create a Linux virtual machine. Then you can play around with Linux in the virtual machine, and if you break something you can spawn a new virtual machine with a fresh Linux install.

To work with Linux virtual machines in Windows, we'll need to install the following programs:

Notepad++
A text editor that works with both Linux style line endings and Windows style line endings.
VirtualBox
Virtualization software which provides an environment for virtual machines.
Vagrant
A command line tool for configuring, creating, and connecting to virtual machines.
Cygwin
A Linux-like command line environment for Windows.

Cygwin

Cygwin Intro

The first thing you need to understand about Cygwin is that it is not a single program. Cygwin is a Linux-like environment that contains many programs and packages. You can add or remove programs and packages from your Cygwin environment using the Cygwin Setup program. The setup.exe file that you download from the Cygwin website is the Cygwin Setup program.

Download Cygwin Setup

Use the button below to download Cygwin Setup.

Download Cygwin

After you've downloaded Cygwin Setup, move the downloaded setup.exe file to a convenient permanent location. Cygwin Setup will cache downloaded packages and package metadata in the same directory as the setup.exe file. Typically, you keep setup.exe in the same place to take advantage of the caching feature. I usually put my setup.exe file in C:\cygwin-setup.

8. Select Packages to Install

Now we're ready to browse for packages and select the Cygwin packages we want to install.

Let's take some time to get familiar with the Cygwin package browser first. Right now we're looking at all Cygwin packages, organized by category.

Click twice to switch to "Pending" view. "Pending" view shows the packages that are currently selected for installation. Hover your mouse over the button for an explanation of the different views.

Right now all of the packages under the "Base" category are selected for installation. The Base category includes all packages required for a minimal Cygwin installation.

In addition to the base packages, we want to install the packages required for Cygwin/X. Cygwin/X is required to run GUI applications in our Linux virtual machine. According to the Cygwin/X website, Cygwin/X requires the following packages:

  • xorg-server
  • xinit
  • openssh

Switch to "Full" view, and type "xorg-server" in the search box.

The first package is the one we want.

Click to cycle through options until it shows the latest version of xorg-server. This selects xorg-server for installation.

Switch to "Pending" view to confirm that xorg-server is selected for installation. Notice that the base packages are hidden since we have typed "xorg-server" into the search filter box.

Click to clear the search filter and see the full list of all pending operations. This unhides the base packages that are selected for installation.

Switch to "Full" view and use the search filter to find xinit and openssh. Click to select xinit and openssh for installation. Switch to "Pending" view to review your pending operations.

Click to execute the pending operations.

Virtual Machine Setup

Cygwin Directory Structure

The rest of this article will assume that the Cygwin root directory is C:\cygwin.

There are a few directories under C:\cygwin that I'd like to point out. Open C:\cygwin in Windows File Explorer.

User Home Directories

Cygwin Setup creates a home directory for each user on your computer under C:\cygwin\home. If your user name is "jdoe", then your Cygwin home directory is C:\cygwin\home\jdoe.

cygdrive Directory

Cygwin Setup creates a C:\cygwin\cygdrive directory. C:\cygwin\cygdrive contains shortcut links to each drive on your computer. For example, C:\cygwin\cygdrive\c is a shortcut to C:\. If you have a D:\ drive, you can access it via C:\cygwin\cygdrive\d. The shortcut links in C:\cygwin\cygdrive allow you to access all files on your computer from within the Cygwin environment.
